🛠 One-Time Setup Guide
Do this once — then just upload CSV daily.
  1. Create a GitHub account at github.com if you don't have one.
  2. Create a new repository named agent-dashboard — set it to Public.
  3. Upload the project files from this folder to your repo (drag & drop on GitHub).
  4. Enable GitHub Pages: Go to repo → Settings → Pages → Source: Deploy from branch → main → / (root).
  5. Generate a token: GitHub → Settings → Developer settings → Personal access tokens → New token → check repo scope.
  6. Paste your token above and enter your repo name, then upload your first CSV.
  7. Share the dashboard link with your associates — it auto-updates on every upload!
